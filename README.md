## Nommage des commits ##
 > Tous les caractères sont autorisés
 
 **Format des messages :** \<type\>(\<portée\>) : \<sujet\>
 > **Exemple :** Refactor(models/User.py) : Ajout de l'attribut "postal_code".
 
 **Type du commit :**
 - **Feat :** Ajout d'une nouvelle fonctionnalité
 - **Fix :** Correction d'un bug
 - **Perf :** Amélioration des performances
 - **Refactor :** Modification qui n’apporte ni nouvelle fonctionalité ni amélioration de performances
 - **Doc :** Rédaction ou mise à jour de documentation
 - **Test :** Ajout ou modification de test(s)

 **Portée du commit :** Indique quelle partie du projet est affectée

 **Sujet du commit :** Description des changements
